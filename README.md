# Fake Sales Data Generator

O objetivo deste programa é gerar massa de dados de vendas para popular banco de dados e datasets para utilização em estudos e demonstrações. 



## Criação do Ambiente virtual

Criação

```bash
python -m venv <nome_ambiente>
```

Ativação(Linux)

```bash
source <nome_ambiente>/bin/activate
```

Desativação

```bash
deactivate
```
